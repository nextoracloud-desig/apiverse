# APIverse

APIverse is a next-generation API marketplace and discovery platform, designed with a premium, futuristic aesthetic.

## Features

- **Global Marketplace**: Explore APIs across categories like AI, Crypto, Weather, and more.
- **Detailed API Insights**: View documentation, pricing, confidence scores, and live code snippets.
- **Interactive Playground**: Test API endpoints directly from the browser.
- **User Dashboard**: Manage API keys, view usage analytics, and save favorite APIs.
- **Premium UI**: "Apple-level" polish with smooth animations and a clean dark mode interface.
